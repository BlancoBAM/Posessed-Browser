# Updating on Windows

## Auto-Update

Posessed Browser checks for updates automatically. When one is available, you'll see a notification in the browser. Click **Relaunch to update** to apply it.

## Manual Update

1. Download the latest installer from [GitHub Releases](https://github.com/BlancoBAM/Posessed-Browser/releases)
2. Run `Posessed-Browser_installer.exe`
3. The installer will update the existing installation in place

## Checking Your Version

Navigate to `chrome://settings/help` in Posessed Browser.
