# Updating Posessed Browser

## Linux

<Steps>
  <Step title="On Lilith Linux">
    Posessed Browser is in the **official Lilith Linux repositories**. System updates include the latest browser:

    ```bash
    sudo apt update && sudo apt upgrade
    ```

    Releases are also published independently to [GitHub Releases](https://github.com/BlancoBAM/Posessed-Browser/releases) and can be installed separately from a system update.

  </Step>
  <Step title="AppImage">
    Download the latest `Posessed-Browser.AppImage` from [GitHub Releases](https://github.com/BlancoBAM/Posessed-Browser/releases) and replace the existing file.
  </Step>
  <Step title="DEB package">
    ```bash
    wget https://github.com/BlancoBAM/Posessed-Browser/releases/latest/download/Posessed-Browser.deb
    sudo dpkg -i Posessed-Browser.deb
    ```
  </Step>
</Steps>

## Windows

<Steps>
  <Step title="Auto-update (if enabled)">
    Posessed Browser will notify you when an update is available. Click
    **Update** in the notification.
  </Step>
  <Step title="Manual update">
    Download the latest installer from [GitHub
    Releases](https://github.com/BlancoBAM/Posessed-Browser/releases) and run
    `Posessed-Browser_installer.exe`. It will update in place.
  </Step>
</Steps>
