---
title: "Update on Linux"
description: "How to update Posessed Browser on Linux"
---

<Steps>
  <Step title="Check for updates">
    Click the **Posessed Browser Feedback** extension (bug icon) in the toolbar, then click **Check for Updates**.

    <Frame>
      <img src="/images/check-for-updates.png" alt="Posessed Browser Feedback extension showing Check for Updates button" />
    </Frame>
  </Step>

  <Step title="Download the latest version">
    Download the Linux build from [GitHub Releases](https://github.com/BlancoBAM/Posessed-Browser/releases).
  </Step>

  <Step title="Install">
    Extract and run the binary. Your settings and data will be preserved.
  </Step>
</Steps>
