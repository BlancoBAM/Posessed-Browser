## v0.39.0

<sub>February 3, 2026</sub>

- **Sync** — Save your browser configuration, agent history, and scheduled tasks across machines
- **App Connector redesign** — New UI makes it simpler to discover, install, and manage connected MCP apps
- **MCP port stability** — Additional fixes for users experiencing port issues across restarts
- **Keyboard shortcuts** — Updated shortcuts to avoid conflicts with European keyboards

---

## v0.38.0

<sub>January 28, 2026</sub>

Bug fix and stability release.

- **MCP port fix on Windows & Linux** — Port now stays consistent across restarts
- **Settings fix** — Fixed `chrome.browser.settings` not working correctly
- **Improved agent** — More reliable and performant agent loop

## v0.37.0

<sub>January 21, 2026</sub>

- **Workflows** — Build reliable, repeatable browser automations with a visual graph builder. [Read more →](/features/workflows)
- **Cowork** — Combine browser automation with local file operations. Research on the web, save reports to files, all in one task. [Read more →](/features/cowork)
