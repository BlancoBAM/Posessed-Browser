# Use with Claude Code / Gemini CLI

Posessed Browser can act as an MCP server, giving AI coding tools like `claude-code` and `gemini-cli` direct control over your browser through 31 tools.

## Setup

### claude-code

Add to your `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "posessed-browser": {
      "command": "npx",
      "args": ["-y", "@browseros/mcp-server@latest"]
    }
  }
}
```

### gemini-cli

Add to your Gemini CLI configuration:

```json
{
  "mcpServers": {
    "posessed-browser": {
      "command": "npx",
      "args": ["-y", "@browseros/mcp-server@latest"]
    }
  }
}
```

### Any MCP Client

The server runs as a local process and exposes a standard MCP interface. Point any MCP-compatible client to the process.

## Available Browser Tools (31 total)

| Category    | Tools                                                 |
| ----------- | ----------------------------------------------------- |
| Navigation  | `navigate`, `back`, `forward`, `reload`               |
| Interaction | `click`, `type`, `scroll`, `hover`, `drag`            |
| Content     | `get_page_content`, `extract_text`, `take_screenshot` |
| Tabs        | `new_tab`, `switch_tab`, `close_tab`, `list_tabs`     |
| Scripts     | `run_javascript`                                      |
| And more    | Forms, cookies, network, storage                      |
